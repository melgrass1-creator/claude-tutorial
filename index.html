📝 第3頁：基本操作與對話技巧

🚀 開始使用 Claude

步驟1：註冊帳號
1. 前往 claude.ai
2. 使用 Google 或 Apple 帳號登入
3. 開始免費使用！

💬 對話基本原則

原則說明
✅ 具體明確說清楚你要什麼
✅ 一次講完把所有背景一次說明
✅ 指定格式告訴他輸出形式
✅ 給範例給範例讓他模仿
❌ 錯誤示範：
"幫我寫一篇關於行銷的文章"

✅ 正確示範：
"幫我寫一篇 800 字的行銷文章，目標讀者是 25-35 歲小資族，語氣活潑專業，用條列式呈現，重點包括：1.社群行銷 2.內容行銷 3.案例一個"

🤖 Claude AI 教學手冊 | 第 3 頁

✨ 第4頁：提示詞黃金法則

🎯 讓 Claude 聽懂你的秘笈

📋 角色扮演法
"你是一位有 10 年經驗的資深業務，幫我..."
📝 分步驟法
"請分三個步驟說明：第一步...第二步...第三步..."
📊 輸出格式法
"請用表格呈現"、"用 JSON 格式"、"條列式說明"
📖 範例引導法
"例如：台北 = 台灣首都，東京 = ?"
⚙️ 條件限制法
"如果是...則..."、"請用繁體中文"、"國中二年級理解程度"

🔄 迭代修正法

• "這不是我想要的，請修改成..."
• "可以更詳細一點嗎？"
• "語氣再輕鬆一點"
• "例子不夠，再多舉幾個"

🎨 第5頁：Artifacts 功能

📦 什麼是 Artifacts？

Artifacts 是 Claude 的即時生成功能，可以立刻產出：
• 網頁（HTML/CSS/JS）
• 文件（Markdown、PDF）
• 圖表
• 程式碼
• 簡報

🚀 如何使用

步驟操作
1打開 claude.ai
2點擊右上角頭像
3選擇 Feature Preview
4開啟 Artifacts

💡 範例指令

"用 HTML 做一個計算機介面，要有加減乘除功能"

"用 Markdown 做一個美味食譜網頁"

"用 CSS 做一個響應式登入表單"
⚠️ 注意：Artifacts 需要付費方案（Pro）才能使用

🔧 第7頁：Claude Code

⌨️ 什麼是 Claude Code？

Claude Code 是 Anthropic 推出的指令列工具 (CLI)
讓 AI 能直接在你的電腦上工作！

🚀 安裝方式（Mac/ Linux）

1. 安裝：
npm install -g @anthropic-ai/claude-code

2. 驗證：
claude --version

💡 基本指令

指令功能
claude開始對話
claude -p "指令"單次執行
claude --help查看幫助
⚠️ 需要 API Key：使用 Claude Code 需要申請 Anthropic API Key

🤖 Claude AI 教學手冊 | 第 7 頁

⚙️ 第8頁：Claude Skills

🎯 什麼是 Claude Skills？

Claude Skills 讓你自訂 AI 的行為模式！
設定一次，永久套用。

📝 如何建立

步驟說明
1打開設定 → Skills
2點擊 Create Skill
3設定名稱與說明
4寫入系統提示詞
5儲存並使用

💡 應用範例

• "每次回覆 email 用正式語氣"
• "寫程式時遵守 ESLint 規範"
• "翻譯時保持原文風格"
• "摘要要點三個重點"
✅ 免費可用：Claude Skills 不需付費！

❓ 第10頁：常見問題與技巧

❓ 常見問題

Q1: Claude 忘記之前的對話？
A: 每個對話是獨立的，切換話題或開新對話即可
Q2: 回覆不滿意？
A: 直接說"不是我要的，請重新回答"，或具體指出哪裡要修改
Q3: 如何上傳檔案？
A: 對話框旁邊有迴紋針 icon，點擊即可上傳

💡 進階技巧

技巧說明
📎 附檔案上傳文件讓 Claude 閱讀分析
🔗 給連結貼上網址讓他讀取網頁
📋 複製結果一鍵複製 AI 回覆
💾 匯出對話可匯出完整對話記錄
