Claude 3 模型家族由 Anthropic 開發，代表了人工智慧在智慧程度、速度與成本效益方面的新標準。其核心理念在於構建可靠、可解釋且安全的系統。

「憲法 AI」引導每一次互動，在不犧牲安全性的前提下，確保提供最有幫助的回應。這是 Claude 與其他模型最顯著的區別之一。

COMPUTER USE (BETA)

Claude 現在能像人類一樣使用電腦：觀察螢幕、移動游標、點擊按鈕以及輸入文字。這項突破性功能讓 AI 能夠執行複雜的多步驟任務。

01 / Perception (感知)

模型即時解讀螢幕截圖，理解 UI 元素、空間關係與視覺提示，甚至能辨識非標準化的介面。

02 / Action (動作)

將自然語言意圖轉化為精確的鍵盤與滑鼠座標指令，在任何應用程式中執行操作。

03 / Reasoning (推理)

具備自我修正行為：如果點擊失敗或出現錯誤，Claude 會重新評估螢幕狀態並嘗試不同路徑。

04 / Sandbox (沙盒)

在安全的隔離容器中運行，確保主機系統的完整性與安全性，並支援 Pro 以上方案使用。

warning

注意：此功能尚處於 Beta 階段，執行時請務必在安全受控的環境下進行。

這是 Anthropic 推出的終端機命令行工具 (CLI)，讓 AI 輔助開發流程變得極其高效。直接在終端機中與 Claude 協作編碼。

常見問題與技巧

Q1: Claude 是否支援繁體中文？

A: 完美支援繁體中文，且語氣較其他模型更為自然、有禮。

Q2: 的用量不足怎麼辦？

A: 建議開啟「新對話」，清理歷史訊息；或考慮升級至 Pro 方案。

Q3: 如何上傳多個檔案？

A: 直接使用對話框旁的「夾帶」icon，支援上傳多達 5 個檔案。
