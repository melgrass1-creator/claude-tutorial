🤖 OpenClaw Tools 與 Skills 教學

Jarvis 的工具箱使用指南

📚 Tools vs Skills 有什麼不一樣？

Tools = 器官：決定 OpenClaw「能不能」做某件事

Skills = 教科書：教 OpenClaw「怎麼組合」Tools 來完成任務

簡單說：沒有 Tools 就沒有手，沒有 Skills 不知道怎麼做事。

⚠️ 重要安全原則

1. exec 開審批

開 exec 的同時設定審批，讓每個命令執行前都要經過你確認。

2. message 只傳給自己

不要讓 OpenClaw 代替你跟任何人溝通——訊息發出去就收不回來了。

3. 最後一哩自己來

結帳、付款、發文——這些不可逆的動作不要交給 AI。

💡 如何開始？

1. 安裝 OpenClaw

2. 設定 openclaw.json

3. 選擇需要的 Tools 和 Skills

4. 開始使用！

{
 "tools": {
  "allow": ["read", "write", "web_search", ...],
  "deny": ["nodes"]
 },
 "approvals": {
  "exec": { "enabled": true }
 }
}
